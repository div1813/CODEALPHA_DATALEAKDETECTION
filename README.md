# 🔐 Data Leak Detection System

A web-based security project that demonstrates **SQL Injection Protection** and **AES-256 Password Encryption** built with pure HTML, CSS, and JavaScript.

---

## 🚀 Live Demo
🔗https://div1813.github.io/CODEALPHA_DATALEAKDETECTION/
---

## 📌 Features

- 🛡️ **SQL Injection Detection** — Blocks malicious inputs in real-time
- 🔒 **AES-256 Encryption** — Passwords are encrypted before being stored
- 📝 **User Registration** — Securely saves encrypted credentials
- 🔑 **User Login** — Decrypts and verifies password on login
- 🚨 **Attack Demo** — Live SQL injection testing with clickable payloads
- 📋 **Attack Log** — Records every test with timestamp and result
